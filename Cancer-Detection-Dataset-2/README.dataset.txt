This dataset is based on the original *Cancer Detection* dataset created by *Breast Cancer* and published on [Roboflow Universe](https://universe.roboflow.com/breast-cancer-4qfmz/cancer-detecion). The dataset was forked and retains a single class representing the **cancer region**, intended for identifying areas affected by cancer.

No additional changes were made to the images or annotations. The dataset is designed for **object detection** tasks focused on cancer localization.
